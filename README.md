## Daftar Program

### 1. function_kompleks.html

Program untuk menghitung nilai akhir mahasiswa menggunakan Function, Parameter, dan Return Value.

Fitur:
- Menggunakan function
- Menggunakan parameter
- Menggunakan return value
- Menggunakan percabangan if else
- Menentukan grade mahasiswa (A, B, C, D, E)

### 2. if_else.html

Program untuk menentukan grade mahasiswa berdasarkan nilai yang diperoleh.

Fitur:
- Menggunakan if
- Menggunakan else if
- Menggunakan else
- Menampilkan hasil grade

### 3. kalkulator.html

Program kalkulator sederhana menggunakan HTML, CSS, dan JavaScript.

Fitur:
- Penjumlahan
- Pengurangan
- Perkalian
- Pembagian
- Menggunakan function JavaScript
- Menggunakan form input
